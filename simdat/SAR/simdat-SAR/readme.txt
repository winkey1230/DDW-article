The file, ScProvince.gal, indicates the spatial adjacency relation among the 180 counties in Sichuan province, China.

The file ,population.txt, is the real population for each county in Sichuan province, China.

ScCoordinate.txt is the longitude and latitude for each county in Sichuan province, China.

A total of six simulated scenarios were set for the SEM and SLM, respectively.
Each simulated dataset file includes a total of 10000 simulated data. Each row has 180 columns corresponding to 180 spatial units which are ranked in the order of population.txt. The first row is the number of  observed cases and the second is the observed explanatory variable, x.
